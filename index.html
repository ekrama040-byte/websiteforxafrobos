About Our Program

We provide structured robotics and programming education for school students. Our program focuses on Python fundamentals, logical thinking, and hands-on projects using educational robots.









What Students Will Learn

  • Python programming basics
  • Robot control and automation
  • Logic building & problem-solving
  • Mini projects and real-world applications
